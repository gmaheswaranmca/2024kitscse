###
Structure of the HTML element:
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>Hello World</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is a web page.</p>
  </body>
</html>

###
what is element?
what is tag?
what is attribute?
explain about anchor element.
In the following tag:
<a href="vendor-list.html">Go Back To Vendor List</a>

Here, 
1. a - the name of the element 
    is the anchor element 
    used to define hyperlink or 
    bookmark 
2. <a ...> is the opening tag 
   <a/> is the closing tag 
3. href="vendor-list.html"
    is the attribute 
    "href" is the name of the attribute 
    "vendor-list.html" is the value of the attribute 

###
Heading level elements:
<h1>Heading Level 1</h1>
<h2>Heading Level 2</h2>
<h3>Heading Level 3</h3>
<h4>Heading Level 4</h4>
<h5>Heading Level 5</h5>
<h6>Heading Level 6</h6>

They define the headings of the document
or section of the document.

Document may be of article, blog, etc. 

###
Paragraph element:
<p>Paragraph is of many sentences.
Each sentence starts with capital letter.
Before "p" element we will have 
line break and after "p" element 
we will have line break when the element 
is processed by HTML parser.

###
Element "UL" - unordered list 
- used to define bullets 

<UL>
    <LI>Item 1</LI>
    <LI>Item 2</LI>
    <LI>Item 3</LI>
    <LI>Item 4</LI>
</UL>

###
Element "OL" - ordered list 
- used to define numberings 
<OL>
    <LI>Item 1</LI>
    <LI>Item 2</LI>
    <LI>Item 3</LI>
    <LI>Item 4</LI>
</OL>
